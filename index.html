作品タイトル
インスタグラムもどき
作品ページ
担当：
全部
使用スキル：
docker, laravel, PostgreSQL, php, html, css
概要：
2019年9月後半の2週間にTeamLabにインターンに行った時に作ったものです。herokuに公開しています。
インスタっぽいものを作ろうというテーマで作成したものです。github認証を用いており、投稿するとgithubのユーザ名とアイコンで投稿されます。未ログインではタイムラインの閲覧しかできません。またいいね機能がついており、いいねの数をクリックするといいねしたユーザーが見えるようになってます。これもログイン時の機能です。あとユーザページには投稿一覧といいね総数が見れるようになっています。いいね機能はajaxを用いて非同期通信しているので特にページ遷移などは起きません。
ちなみに自分がテストした以外誰にも使ってもらってないので登録ユーザは私だけです...
現在確認しているバグとしていいねを連打すると数値が無限に上げるもしくは下げることができます。要するに連打時の処理ができてないです。気が向いたらいいねの数値が変動するまでクリック処理を受け付けないように直します。

作品タイトル
アルこ〜る
担当：
サーバサイドとチャットシステム
使用スキル：
nginx, nodejs, html, css, webRTC, websocket
概要：
2018年のyahooハッカソンに参加した時に作った作品です。研究室のメンバーと合わせて4人で参加しました。期間は2週間、テーマは「コール&レスポンス」 ということで酒飲み同士が繋がれるウェブ通話アプリ「アルこ〜る(アルコール + call)」を開発。 制作理由は「俺が欲しかったから」ここ重要。
システム想定は「居酒屋を家で再現する」ことで、通話は1グループ最大4人(居酒屋の1テーブルを想定)、チャットはサイトにアクセスしている全員に通じる(居酒屋で他の席から聞こえてくる会話みたいな想定)ようになっている。
自身の担当範囲はサーバ関連と通話や文字でのチャット機能の実装をした。 nginxとnode.jsサーバをたてた。通信はsocketを使ってP2P通信で行ない、音声通話についてはwebRTCを使った。最初は映像付でやっていたけどプライバシーとかの方面考えるといらなくない？となったので撤去。ついでにユーザ機能もいらない(もといわからない&時間ない)となって毎回匿名で接続するようになった。映像ON/OFF機能も作ったけどそもそもONいらないと言われたので撤去。悲しい。そのくせよく考えたら音声ミュート機能作ってなかったなって今これを書きながら思いました。あとはLINE風チャットを作った。
結局ユーザの概念を消したせいで闇鍋斎藤さんみたいになった。特に公開などはしてないが手直しをして公開する日がくるかもしれない...

Watashi ha Onigiri desu(私はおにぎりです)
学部3年の時に研究室に来ていた海外留学生と一緒に作ったもの。Unityとアマゾンダッシュボタンを使った。IoTボタンを使ってゲームを作る、という課題のもと作った。2チームに分かれて行うゲームであり、IoTボタンをパワーアップアイテムとして考えり、片方のチームが現実世界にIoTボタンを隠し、もう片方のチームが探し出す。見つけたボタンを押せばUnity側のゲームのキャラクターが強くなる、もしくは有利に働くイベントが発生するといったものである。これを各チーム交互にやってスコアを競うといった内容。
IoTボタンで何かを操作するにはラグが発生していまいち思ったようにならないため、このように現実とリンクさせたゲームを作ることになった。個人的にこのアイデアはすごく気に入っている。 「あいちゃれ」という学内のコンテスト(今はもう開催されてないらしい)で入賞した。動画はリンクがなかった...

昔twitterのbot作ってなんか作りたいなと思った時に考えていたもの。その時は何つぶやくか考えてなかったし、知識も全然なかったので未完成のままだった。

授業でやったUnityを使ったゲーム(カケルプロジェクト)
学部3年の時にUnityとHMDを使ってVRゲームを作成した。「カケル」がテーマであり、「翔」や「書ける」など、ギャグでもいいので「カケル」を使うのが題材だった。班員は確か7名？で自分はイベント発火などのスクリプトの作成担当した。操作キャラの座標や経過時間でのフラグ管理が基本だった。あと非現実的なことを題材にしてたため空の色を変えたりホラーっぽく光源や音源をいじったりもした。10班中3位だった記憶がある。

授業でやったホームページ作成
こちらも学部3年の時に授業でやった内容。IT用語を調べて、その内容を紹介するホームページを作成すると言った内容。当班の課題は「プロジェクションマッピング」だった。調べる部分は置いといて、ホームページではjQueryを使って画像の自動スクロールや自動切り替えとかを作った記憶がある。あと発表時にスマホの映像の上に4枚の透明フィルムを置いてスマホの上に立体映像を出した記憶がある。
